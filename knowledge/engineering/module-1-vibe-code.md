# Module 1 — Vibe Code: Zero to Hero

## What This Module Covers

This module teaches the engineering fundamentals every non-technical founder needs to vibe code like an engineer instead of a beginner. Without these fundamentals, LLMs paint themselves into a corner fast. With them, you can achieve what the best engineers are achieving — moving 100x, 1000x faster than ever before.

This module covers:

1. **The three layers** — Every app ever built runs on exactly three layers. Understanding what they are and how they talk to each other is the foundation of everything.

2. **Schema** — The single concept that makes Claude 10x more useful. Design your data structure before writing a single prompt and Claude will never lose the thread.

3. **How to structure your data** — Tables, fields, types, relationships. One-to-many. Many-to-many. Foreign keys. The Kevin method for thinking about it without any technical background.

4. **APIs, auth, state, and environment** — The remaining concepts you need to understand to build production-ready software.

5. **The only stack you need** — TypeScript, Next.js, Hasura, Turso. Why these tools and no others. Why this stack takes you from zero to $1M without touching the foundation.

6. **Hosting for under $10 a month** — Turso, Hasura Cloud, and Vercel each have generous free tiers. Real production apps running for practically nothing.

7. **Debugging** — How to get your LLM to unshit itself. Logs first. Fix second. Never guess.

8. **Long tasks** — How to set Claude on a mission and not let it stop until it wins. Test-driven development for vibe coders. The closed loop.

## The Core Principle

Frontend shows. Backend thinks. Database remembers.

Get that right and every prompt you write to Claude will be dramatically more effective. Get it wrong and no amount of prompting gets you out of the corner you've painted yourself into.
