# Ad Templates

Reusable ad copy templates for Reddit, Meta, and Google Search.

Templates will be added here as campaigns are developed and tested. Each template includes:
- Platform (Reddit / Meta / Google)
- Ad type (awareness / conversion / retargeting)
- Headline
- Body copy
- CTA
- Target audience
- Historical performance notes (if available)

See /skills/marketing/reddit-ad-launch.md, meta-ad-launch.md, and google-search-ad-launch.md for the skills that use these templates.
