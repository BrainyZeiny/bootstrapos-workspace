# Email Templates

Reusable cold outbound email templates organized by use case.

Templates will be added here as sequences are developed and tested. Each template includes:
- Use case (cold outreach / follow-up / affiliate / nurture)
- Subject line variations
- Body copy
- CTA
- Target persona
- Historical performance notes (reply rate, if available)

See /skills/marketing/mass-email-at-scale.md and /skills/marketing/cold-outbound-infrastructure.md for the skills that use these templates.
