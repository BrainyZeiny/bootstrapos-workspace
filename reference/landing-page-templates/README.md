# Landing Page Templates

Reusable landing page structures for different offer types.

Templates will be added here as pages are built and conversion-tested. Each template includes:
- Use case (SaaS free trial / lead gen / waitlist / sales call booking)
- Section structure (hero, problem, solution, proof, CTA)
- Copy frameworks
- Component breakdown
- Historical conversion rate notes (if available)

See /skills/marketing/landing-page-builder.md for the skill that builds these pages.
